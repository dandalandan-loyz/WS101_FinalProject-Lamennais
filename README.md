# AI Co-Pilot Log

## Student Information
- **Student Name:** Dan Louise J. Baluntong
- **Section:** BSIT-3B
- **Project Topic:** La Mennais Community Website Development

## Project Overview
This project is a multi-page website for the La Mennais Community of San Jose de Buenvista. It presents the community's identity, services, gallery, contact information, and other important content through a clean and responsive interface.

## Framework and Technology Used
- **Frontend Structure:** HTML5
- **Styling:** Custom CSS3
- **Interactivity:** Vanilla JavaScript
- **Typography:** Google Fonts (`Playfair Display` and `Inter`)
- **Media Assets:** Local image assets and selected online placeholders during development
- **Framework Used:** No front-end framework was used. The website was built using a custom static structure with vanilla HTML, CSS, and JavaScript.

## How I Used AI in Developing the Project
AI was used as a development assistant throughout the creation and refinement of the website. I used it to help generate structure, improve design ideas, debug layout issues, fix image and file path problems, and refine sections that needed correction. The AI responses helped speed up the workflow, but I still reviewed the output, adjusted the code, tested changes, and made the final implementation decisions myself.

## AI Prompt Log

### 1. Initial Website Structure and Code Generation
- **AI Tool Used:** ChatGPT
- **Prompt:**
  > give me the full code of this website design that i attached screenshots images as my reference from top to bottom. make sure to separate the html, css, and js files. i also attach my file path screenshot
- **How I Used It:**
  I used the generated output as a starting guide for organizing the website's core sections such as the header, hero, services, gallery, and footer. After that, I manually refined the layout, improved responsiveness, and adjusted the content and visual identity to better match the La Mennais Community.

### 2. Hero Image and File Path Debugging
- **AI Tool Used:** ChatGPT
- **Prompt:**
  > i want to add an image behind the hero section
  > fix my file path and make sure the image appears properly
- **How I Used It:**
  I used AI guidance to correctly apply background images through CSS and fix file path issues. This helped me understand how to use `background-image`, `background-size: cover`, and `background-position: center` so that hero images would display properly.

### 3. UI/UX Design and Layout Refinement
- **AI Tool Used:** ChatGPT
- **Prompt:**
  > create a modern and clean website design for a religious community website with sections like services, gallery, and contact
- **How I Used It:**
  The response helped shape the site's visual hierarchy and section flow. I used the suggestions to improve spacing, card layouts, typography, and color consistency while keeping the design respectful and appropriate for a faith-centered community website.

### 4. Image Integration and Content Mapping
- **AI Tool Used:** ChatGPT
- **Prompt:**
  > add images to service cards and gallery using proper file names
  > use placeholder or online images if local images are not available
- **How I Used It:**
  I used this to map images into service cards, gallery items, and feature sections. During development, some local images were not yet available, so online placeholders were temporarily used to complete the layout and maintain visual balance.

### 5. Interactive Elements and Media Integration
- **AI Tool Used:** ChatGPT
- **Prompt:**
  > make the video placeholders clickable and redirect to youtube
  > make the video cover visible
- **How I Used It:**
  AI helped me implement clickable video areas that link to YouTube content and improve the visual presentation of the media section. This made the page feel more interactive and realistic. But i applied embedded link in youtube later on.

### 6. Map Integration
- **AI Tool Used:** ChatGPT
- **Prompt:**
  > make the map placeholder clickable and redirect to google maps
- **How I Used It:**
  I used the suggestion to make the map section more functional by linking it to the actual Google Maps location and using a map preview image for better usability. Applied embedded link on maps later on.

### 7. Gallery Layout and Styling Improvements
- **AI Tool Used:** ChatGPT
- **Prompt:**
  > fix the gallery layout and improve alignment using design principles
- **How I Used It:**
  This helped me improve the gallery using grid and masonry layout ideas, more consistent spacing, and better visual hierarchy. I adjusted the implementation so it fit the rest of the site's styling.

### 8. Leadership Section Image Cleanup
- **AI Tool Used:** ChatGPT / Codex
- **Recent Prompt:**
  > You are editing a real project. Apply the fix DIRECTLY and rewrite the file.

  I used AI to directly fix the leadership cards in `about.html` by ensuring each `.leader-photo` container kept only one image. This resolved the repeated-image issue and restored the intended card structure.

### 9. Gallery Preview Image Fix
